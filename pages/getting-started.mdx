# Getting Started with Node.js and TypeScript

Welcome to the getting started guide! This section is designed to help you set up your development environment and get started with Node.js and TypeScript.

Here's what you'll learn in this section:

1. [Setting Up Node.js](/getting-started/setting-up-nodejs): In this guide, you'll learn how to install Node.js and create a new Node.js project.

2. [Setting Up TypeScript](/getting-started/setting-up-typescript): In this guide, you'll learn how to install TypeScript and create a TypeScript configuration file.

3. [Understanding tsconfig.json](/getting-started/tsconfig-explained): In this guide, you'll learn about the `tsconfig.json` file and the various options that you can use to configure the TypeScript compiler.

By the end of this section, you'll have a solid understanding of how to set up a Node.js project with TypeScript. You'll also understand how to configure the TypeScript compiler to suit the needs of your project.

So let's get started!
